Overview

Site Name

Digital Monopoly Assistant

Purpose

The purpose of Digital Monopoly Assistant is to replace physical paper money and dice used in Monopoly games. This web app makes gameplay faster and more convenient by providing digital dice rolling, player money tracking, and a transaction history log to keep the game organized and fair.

Audience

This tool is designed for Monopoly players of all ages who play the physical board game and want a modern, cleaner, and easy-to-use digital companion. It is especially helpful for families, students, and friend groups.

Dynamic Elements (JavaScript)

  • Random dice roll with visual updates
  • Player cards created dynamically (starts with 2 players, Add Player button)
  • Player cards store: name, money, status
  • Editable money amounts with buttons
  • +200 button for passing GO
  • Transaction log that updates live
  • Conditional "Bankrupt" display if money <= 0
  • Players stored as objects inside an array
  • Array methods used for updates and rendering
  • Modules: dice.js, players.js, history.js, main.js
